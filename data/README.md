# Data

## `raw/mango_dmc_nir.csv` (not committed — fetch it)

The public mango dry-matter benchmark this project is built on:

- **Source:** Anderson, N.T., Walsh, K.B. and Subedi, P.P. (2020) *Mango DMC and NIR spectra*. Mendeley Data, doi:10.17632/46htwnp833. Licence: CC BY 4.0.
- **Contents:** 11,691 near-infrared scans of intact mango (309–1140 nm) with laboratory oven-dry dry-matter values, across 10 cultivars and 4 seasons.
- **Size:** ~34 MB, so it is fetched rather than stored in the repository.
- **SHA-256:** `20330fdd05d6c7c6d1ee948f765e7fb32a16b471f40ec04c3cba64ea418b25c5`

Get it with:

```
python scripts/fetch_data.py
```

which downloads the file and checks it against the hash above, so every result is computed on the exact copy pinned here.

## `farm/aamparakh_farm_readings.csv`

The on-orchard validation set. Until real fruit are measured, this holds a physically-grounded
placeholder (every row flagged `SYNTHETIC_PLACEHOLDER`), regenerated by `src/aamparakh/farm.py`.
Replace it with real Alphonso/Kesar readings using `docs/FARM_DATA_COLLECTION.md`; the analysis
then uses your data automatically.
